# postgre_sql_project
This project is an end-to-end SQL data analysis case study built using three core datasets - Books, Customers and Orders. It simulates a real online bookstore business and demonstrates how SQL is used to analyze sales performance, customer behavior, and product trends.

PROJECT OBJECTIVE
The goal of this project is to practice and showcase industry-level SQL skills by :
1. Deisgning a relationla database
2. Importing raw .csv files
3. Cleaning and exploring structured data
4. Writing analytical SQL queries to answer business questions
5. Presenting insights that support decision-making

DATASET DESCRIPTION 
1. Boooks.csv
contains books information such as:
Book_id, Title, Author, Genre, Published_year, Stock, Price.

2. Customer.csv
 contains customers information like:
Customer_id, Name , E-Mail, Phone, City, Country.

3. Orders.csv
captures order-related information:
Order_id, Customer_id, Book_id, Order_date.Quantity, Total_Amount.
